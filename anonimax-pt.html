📖 Guia Completo — Anonimax
🔒 Como funciona o Anonimax?
O Anonimax é um mercado de classificados 100% anônimo. Nenhum nome, telefone ou e-mail é necessário.
1
Crie seu Anonimax ID
Registre seu ID Session + endereço Polygon → receba um código único ANX-XXXXX e um QR Code do seu perfil.
2
Publique seus anúncios
Crie anúncios pagando R$ 4,90. Seu Anonimax ID aparece no anúncio — não seu nome ou telefone.
3
Seja encontrado pelo ID ou QR Code
Qualquer pessoa busca ANX-XXXXX na barra de pesquisa, ou escaneia seu QR Code para acessar seu perfil diretamente.
4
Converse via Session, receba em BRZ
Botão "Conversar no Session" abre o app automaticamente. Pagamento direto em BRZ no seu endereço Polygon.
🔒 Privacidade total: Comunicação criptografada via Session. Pagamento direto na blockchain Polygon. Nenhum intermediário.
